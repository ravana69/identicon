<h4>Raw</h4>
<p>Identicon toString method accepts a boolean argument that when set to true returns raw image data which is slightly smaller. In the case of SVG this is still safe to use in a data URI provided that element attribute is double quoted.</p>
<pre>
var data = new Identicon(hash, options).toString(true);
document.write('&lt;img src="data:image/svg+xml;utf8,' + data + '"&gt;');</pre>
